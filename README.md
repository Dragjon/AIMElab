# AIMElab
Your personal math olympiad practice site with full AIME problem sets and solutions from 1993-2024

# How it works
All the AIME problems were taken from this [dataset](https://www.kaggle.com/datasets/hemishveeraboina/aime-problem-set-1983-2024). The **asymptote** api compile server is hosted with [render](https://render.com/) and is also part of another github repo [here](https://github.com/Dragjon/asymptote-renderer). The platform allows u to try out random AIME problems, unseen problems as well as redo problems u got wrong last time. There is a also a practice functionality which allows u to customise your own practice sessions. All data from the practice is saved locally in your device's localstorage (currently) and is completely private. I am considering scaling this to make it public in the future to provide a leaderboard functionality maybe. You can also browse past AIME problems using the browse functionality and attempt specific problems. The statistic page allows u to view detailed statistics about your performance over time, streaks as well as overall accuracy by problem number. You can also revisit your past problems through the history tab. There are also various acessibility/themes settings for u to explore :) Have funn
